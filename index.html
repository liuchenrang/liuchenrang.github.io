服务器无法rsa登陆

发表于 2017-11-06 更新于 2021-11-06 分类于 linux

ssh 无法登陆服务器，调试方法

被rsa 无法登陆坑了很久 无意中发现了别人提供了一个rsa方式无法ssh登陆的调试大杀器 特此记录下

服务器端 开启sshd 调试服务器

/usr/sbin/sshd -d -p 2222 -d 参数应该是调试状态 -p 是端口号

ssh -vvv root@www.phpyiqiwan.com -p 2222 客户端连接采用 -vvv 显示冗余信息 即是调试信息

这是这样仍然无法登陆

要下ssh rsa 登陆成功 必须保持 自己的家目录 必须都是自己的文件 只有自己可以读写

所以 必须这样 chown -Rf root.root /root 执行下 方可
